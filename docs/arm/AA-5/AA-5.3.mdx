<p>[An <code><a href="../AA-5/AA-5.3#S0175">if_statement</a></code> selects for execution at most one of the enclosed <code>sequences_of_statements</code>, depending on the (truth) value of one or more corresponding <code><a href="../AA-4/AA-4.5#S0150">condition</a></code>s.] </p>

#### Syntax

<p><code>if_statement</code><a id="S0175"></a><code> ::= </code>    <strong>if</strong> <code><a href="../AA-4/AA-4.5#S0150">condition</a></code> <strong>then</strong>      <code><a href="../AA-5/AA-5.1#S0166">sequence_of_statements</a></code>   {"{"}<strong>elsif</strong> <code><a href="../AA-4/AA-4.5#S0150">condition</a></code> <strong>then</strong>      <code><a href="../AA-5/AA-5.1#S0166">sequence_of_statements</a></code>{"}"}   [<strong>else</strong>      <code><a href="../AA-5/AA-5.1#S0166">sequence_of_statements</a></code>]    <strong>end</strong> <strong>if</strong>;</p>
<p><em>Paragraphs 3 and 4 were deleted.</em> </p>

#### Dynamic Semantics

<p>{"{"}<em>{"AI05-0264-1"}</em>{"}"} For the execution of an <code><a href="../AA-5/AA-5.3#S0175">if_statement</a></code>, the <code><a href="../AA-4/AA-4.5#S0150">condition</a></code> specified after <strong>if</strong>, and any <code><a href="../AA-4/AA-4.5#S0150">condition</a></code>s specified after <strong>elsif</strong>, are evaluated in succession (treating a final <strong>else</strong> as <strong>elsif</strong> True <strong>then</strong>), until one evaluates to True or all <code><a href="../AA-4/AA-4.5#S0150">condition</a></code>s are evaluated and yield False. If a <code><a href="../AA-4/AA-4.5#S0150">condition</a></code> evaluates to True, then the corresponding <code><a href="../AA-5/AA-5.1#S0166">sequence_of_statements</a></code> is executed; otherwise, none of them is executed. </p>
<Admonition type="note">
<p><strong>Ramification: </strong>The part about all evaluating to False can't happen if there is an <strong>else</strong>, since that is herein considered equivalent to <strong>elsif</strong> True <strong>then</strong>. </p></Admonition>


#### Examples

<p><em>Examples of if statements:</em> </p>
<CodeBlock>
if Month = December and Day = 31 then{"\n"}   Month := January;{"\n"}   Day   := 1;{"\n"}   Year  := Year + 1;{"\n"}end if;{"\n"}
</CodeBlock>

<CodeBlock>
if Line_Too_Short then{"\n"}   raise Layout_Error;{"\n"}elsif Line_Full then{"\n"}   New_Line;{"\n"}   Put(Item);{"\n"}else{"\n"}   Put(Item);{"\n"}end if;{"\n"}
</CodeBlock>

<CodeBlock>
if My_Car.Owner.Vehicle /= My_Car then            --  see 3.10.1{"\n"}   Report ("Incorrect data");{"\n"}end if;{"\n"}
</CodeBlock>
