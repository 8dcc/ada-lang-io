#### Static Semantics

<p>The following <code><a href="../AA-10/AA-10.1#S0289">library_unit_renaming_declaration</a></code>s exist: <br /></p>

<CodeBlock>
with Ada.Unchecked_Conversion;{"\n"}generic function Unchecked_Conversion renames Ada.Unchecked_Conversion;{"\n"}
with Ada.Unchecked_Deallocation;{"\n"}generic procedure Unchecked_Deallocation renames Ada.Unchecked_Deallocation;{"\n"}
with Ada.Sequential_IO;{"\n"}generic package Sequential_IO renames Ada.Sequential_IO;{"\n"}
with Ada.Direct_IO;{"\n"}generic package Direct_IO renames Ada.Direct_IO;{"\n"}
with Ada.Text_IO;{"\n"}package Text_IO renames Ada.Text_IO;{"\n"}
with Ada.IO_Exceptions;{"\n"}package IO_Exceptions renames Ada.IO_Exceptions;{"\n"}
with Ada.Calendar;{"\n"}package Calendar renames Ada.Calendar;{"\n"}
with System.Machine_Code;{"\n"}package Machine_Code renames System.Machine_Code; -- If supported.{"\n"}

</CodeBlock>
<Admonition type="aarm" aarm="discussion" title="Discussion: ">
<strong></strong>{"{"}<em>{"AI05-0004-1"}</em>{"}"} These library units correspond to those declared in Ada 83, which did not have the child unit concept or the parent package Ada. <br /></Admonition>


#### Implementation Requirements

<p>The implementation shall allow the user to replace these renamings. <br /></p>
