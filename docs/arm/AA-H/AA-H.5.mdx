import Admonition from "@theme/Admonition";

<p>{"{"}<em>{"AI95-00305-01"}</em>{"}"} {"{"}<em>{"AI12-0267-1"}</em>{"}"} <br />The following <code><a href="../AA-2/AA-2.8#S0019">pragma</a></code> requires an implementation to detect potentially blocking operations during the execution of a protected operation or a parallel construct. <br /></p>


#### Syntax

<p>{"{"}<em>{"AI95-00305-01"}</em>{"}"} <br />The form of a <code><a href="../AA-2/AA-2.8#S0019">pragma</a></code> Detect_Blocking is as follows: <br /></p>

<p>  <strong>pragma</strong> Detect_Blocking; <br /></p>


#### Post-Compilation Rules

<p>{"{"}<em>{"AI95-00305-01"}</em>{"}"} <br />A <code><a href="../AA-2/AA-2.8#S0019">pragma</a></code> Detect_Blocking is a configuration pragma. <br /></p>


#### Dynamic Semantics

<p>{"{"}<em>{"AI95-00305-01"}</em>{"}"} {"{"}<em>{"AI12-0247-1"}</em>{"}"} {"{"}<em>{"AI12-0267-1"}</em>{"}"} <br />An implementation is required to detect a potentially blocking operation that occurs during the execution of a protected operation or a parallel construct defined within a compilation unit to which the pragma applies, and to raise Program_Error (see 9.5). <br /></p>


#### Implementation Permissions

<p>{"{"}<em>{"AI95-00305-01"}</em>{"}"} {"{"}<em>{"AI12-0267-1"}</em>{"}"} <br />An implementation is allowed to reject a <code><a href="../AA-10/AA-10.1#S0286">compilation_unit</a></code> to which a pragma Detect_Blocking applies if a potentially blocking operation is present directly within an <code><a href="../AA-9/AA-9.5#S0260">entry_body</a></code>, the body of a protected subprogram, or a parallel construct occurring within the compilation unit. <br /></p>

<Admonition type="aarm" aarm="note" title="Note: ">
NOTE 1   {"{"}<em>{"AI95-00305-01"}</em>{"}"} {"{"}<em>{"AI12-0442-1"}</em>{"}"} <br />An operation that causes a task to be blocked within a foreign language domain is not defined to be potentially blocking, and is unlikely to be detected. <br /></Admonition>


#### Extensions to Ada 95

<Admonition type="aarm" aarm="note" title="Note: ">
{"{"}<em>{"AI95-00305-01"}</em>{"}"} <br />Pragma Detect_Blocking is new. <br /></Admonition>


#### Extensions to Ada 2012

<Admonition type="aarm" aarm="note" title="Note: ">
{"{"}<em>{"AI12-0267-1"}</em>{"}"} <br />Pragma Detect_Blocking now applies to parallel constructs as well as protected actions. <br /></Admonition>
